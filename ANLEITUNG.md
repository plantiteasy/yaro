# Сайт памяти Ярослава — инструкция

Весь сайт — один файл `index.html`. Два языка (русский / немецкий), переключатель сверху, выбор запоминается. Фотографии лежат в папке `photos/`.

## Что уже заполнено

- Имя, даты жизни (14.01.1990 – 30.06.2026), портрет
- Прощание: церемония 10.07.2026 в 10:00 (Ramsaier Bestattungen, Stuttgart-Vaihingen) и поминальный обед в 13:00 (Stuttgart-Degerloch), с картами
- Сбор PayPal от Николь
- Галерея «Моменты», цитаты «Его словами», раздел о его деле
- Кнопка «Сообщить о присутствии» — пока открывает письмо на email Дениса

## Как опубликовать сайт (два шага в настройках GitHub)

Публикация настроена автоматически (workflow `.github/workflows/pages.yml`). Нужно только:

1. **Переименовать репозиторий (по желанию):** Settings → General → Repository name → например `yaroslav` → Rename. Тогда адрес будет `https://plantiteasy.github.io/yaroslav/`.
2. **Сделать репозиторий публичным:** Settings → General → вниз до Danger Zone → Change repository visibility → Make public.

После этого публикация запустится сама (или попросите Клода перезапустить). Сайт появится по адресу `https://plantiteasy.github.io/<имя-репозитория>/` через 1–2 минуты.

## Как менять данные позже

Откройте `index.html` → в самом низу блок `const CONFIG = {`:

| Поле | Что вписать |
|---|---|
| `whatsapp` | Номер Дениса для подтверждений: межд. формат без `+`, напр. `"4917612345678"` — кнопка переключится с email на WhatsApp |
| `album` | Ссылка на общий альбом Google Photos (альбом → Поделиться → «Разрешить добавлять фото») |
| `guestbookForm` | Ссылка на Google Форму для слов памяти |
| `photo` | Другой файл портрета, напр. `"photos/portrait.jpg"` |

Редактировать можно прямо на GitHub: открыть файл → карандаш → внести правку → Commit. Через минуту-две сайт обновится сам.

## Свечи памяти: как одобрять

Каждая «зажжённая» гостем свеча приходит письмом на email Дениса (**первое письмо от formsubmit.co попросит один раз нажать кнопку подтверждения — обязательно нажмите**). На стене свечей появляются только одобренные записи.

Чтобы одобрить свечу: откройте файл `candles.json` на GitHub → карандаш → добавьте запись по образцу → Commit. Через 1–2 минуты свеча загорится на сайте:

```json
{
  "candles": [
    { "name": "Имя Фамилия", "text": "Слова памяти (можно убрать)", "date": "07.07.2026" }
  ]
}
```

Можно просто переслать письмо Клоду со словом «одобряю» — он внесёт сам.

**Право одобрять у других людей**: Settings → Collaborators → Add people → указать GitHub-аккаунт Николь или Игоря. Пока право есть только у владельца (у вас).

## Присланные гостями фото и видео: как опубликовать

Гости шлют файлы в WhatsApp Дениса (кнопка на сайте). Одобренное публикуется так:

1. GitHub → папка `photos/` → **Add file → Upload files** → загрузить файл (видео тоже сюда; до 100 МБ)
2. Открыть `gallery.json` → карандаш → добавить строку в список `items`:
```json
{ "src": "photos/имя-файла.jpg", "cat": "free", "by": "Имя приславшего" }
```
Для видео добавить `"type": "video"`. Категории: `work` (на кухне), `free` (свободное время), `just` (просто Ярик).
3. Commit → через 1–2 минуты на сайте.

Либо просто переслать файл Клоду со словом «да» — он сделает всё сам.

## Новые фотографии в галерею

1. Загрузите файлы в папку `photos/` (Add file → Upload files)
2. В `index.html` в блоке `<div class="gallery-grid">` добавьте строку `<img src="photos/имя-файла.jpg" alt="" loading="lazy">`
