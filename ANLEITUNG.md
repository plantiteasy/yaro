# Сайт памяти Ярослава — как заполнить и опубликовать

Весь сайт — один файл `index.html`. Он работает на двух языках (русский / немецкий), язык переключается кнопками сверху и запоминается.

## 1. Заполнить данные

Откройте `index.html`, найдите в самом низу блок `const CONFIG = {` и впишите:

| Поле | Что вписать |
|---|---|
| `fullName` | Полное имя, напр. `"Ярослав Иванов"` (пусто = просто «Ярослав») |
| `dates` | Годы жизни, напр. `"12.03.1990 – 28.06.2026"` |
| `photo` | Путь к фото, напр. `"photos/yaroslav.jpg"` — создайте папку `photos/` и положите файл |
| `whatsapp` | Номер Дениса для подтверждений: межд. формат без `+`, напр. `"4917612345678"` |
| `paypal` | Ссылка на сбор Николь (PayPal) |
| `album` | Ссылка на общий альбом Google Photos (создать альбом → Поделиться → включить «Разрешить добавлять фото») |
| `guestbookForm` | Ссылка на Google Форму для слов памяти (необязательно) |
| `email` | Email для воспоминаний, если формы нет |

Пока поле пустое — соответствующая кнопка/раздел просто не показывается. Ничего не сломается.

## 2. Опубликовать (GitHub Pages, бесплатно)

1. В репозитории на GitHub: **Settings → Pages**
2. **Source**: Deploy from a branch → выбрать `main` (после слияния PR), папка `/ (root)`
3. Через 1–2 минуты сайт будет доступен по адресу `https://<логин>.github.io/KASSE/`

Совет: репозиторий можно переименовать (Settings → General → Rename), например в `yaroslav`, тогда адрес будет красивее: `.../yaroslav/`. Позже можно подключить и собственный домен.

## 3. Сбор подтверждений (кто придёт)

Кнопка «Ответить в WhatsApp» открывает чат с Денисом с уже заполненным шаблоном:

> Приду на церемонию (10:00): да/нет
> Приду на поминальный обед (13:00): да/нет
> Количество человек: …
> Имя, фамилия: …

Остаётся только вести список имён и передать его Николь, как договорились.

Если людей будет много и хочется автоматики — создайте Google Форму (вопросы: имя, церемония да/нет, обед да/нет, кол-во человек), вставьте ссылку в `guestbookForm`-подобное поле или пришлите её сюда — добавлю отдельной кнопкой. Ответы сами собираются в таблицу.
